# Contributing

## How to Contribute
1. Fork the repository and create a feature branch.
2. Make your changes, test locally.
3. Submit a Pull Request with a clear description.

## Guidelines
- Follow PEP8 for Python code.
- Keep the frontend in plain HTML/CSS/JS (no frameworks).
- Test API changes manually before submitting.
- Update docs if you change any features.

## Project Structure
```
backend/      — Flask app, RAG pipeline, config
frontend/     — HTML, CSS, JS (no build step)
docs/         — Documentation
```

## Code of Conduct
Be respectful and constructive. No harassment or discrimination.
